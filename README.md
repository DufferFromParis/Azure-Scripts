# Azure-Scripts
Scripts I used during some projects:

  - [Migrate-AzKeyVault](https://github.com/DufferFromParis/Azure-Scripts/tree/main/Migrate-AzKeyVault) : Powershell script to help migrating Key Vaults from the access policy model to RBAC.
  - [Update-AzVMExtensions](https://github.com/DufferFromParis/Azure-Scripts/tree/main/Update-AzVMExtensions) : Powershell script to update VM Extensions to the latest version on all VMs in the given subscription.
